# 🇰🇪 Magical Kenya | Cultural & Tourist Hub 2026

An immersive, high-end web portal designed to provide international tourists with the essential tools and cultural insights needed for a seamless Kenyan safari experience.

![Kenya Safari Banner](https://images.unsplash.com/photo-1516426122078-c23e76319801?q=80&w=1200)

## ✨ Key Features

* **📍 Interactive Safari Map:** Built with `Leaflet.js`, allowing users to explore key destinations like the Maasai Mara, Diani Beach, and Mt. Kenya with real-time descriptions.
* **🗣️ Swahili Voice Assistant:** Uses the `Web Speech API` to help travelers hear and learn essential phrases for their journey.
* **🏺 3D Cultural Gallery:** An embedded `A-Frame` (WebVR) experience showcasing Kenyan artifacts in a 3D space.
* **🌙 Modern Ethno-UI:** A high-performance, dark-themed interface utilizing Glassmorphism and "Midnight Kenya" aesthetics.
* **🛠️ Traveler Toolkit:** Integrated local time (EAT) and a budget estimator for KES currency conversion.

## 🚀 Tech Stack

* **Frontend:** HTML5, CSS3 (Advanced Flexbox/Grid), Modern JavaScript (ES6+)
* **Libraries:** * [Leaflet.js](https://leafletjs.com/) - Interactive Maps
    * [A-Frame](https://aframe.io/) - Virtual Reality/3D Rendering
    * [Google Fonts](https://fonts.google.com/) - Plus Jakarta Sans
* **APIs:** Web Speech Synthesis API

## 🛠️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/cultureweb.git](https://github.com/your-username/cultureweb.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd cultureweb
    ```
3.  **Launch Live Server:**
    * Open the folder in **VS Code**.
    * Click **"Go Live"** in the bottom right corner (requires Live Server extension).
